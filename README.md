# 💰 Expense Manager (Money Trail)

## 📌 Project Description
Expense Manager is a Java-based web application that helps users track their daily income and expenses efficiently. It provides features to manage accounts, categories, subcategories, and generate insights into financial activities.

---

## 🚀 Features
- User Registration & Login 🔐
- Dashboard with summary 📊
- Add / Update / Delete Expenses 💸
- Add / Manage Income 💰
- Category & Subcategory Management 🗂️
- Account Management 🏦
- Role-based access (Admin/User) 👤
- Secure session handling 🔒

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot
- **Frontend:** JSP, HTML, CSS
- **Database:** MySQL
- **Build Tool:** Maven
- **IDE:** Spring Tool Suite (STS)
